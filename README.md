# JWT - JSON Web Token

It is defined in RFC 7519

JWTs can be signed using a secret (with the HMAC algorithm) or a public/private key pair using RSA.

JSON Web Tokens consist of three parts separated by dots (.), which are:

Header
Payload
Signature
